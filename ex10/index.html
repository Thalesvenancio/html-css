listas ordenadas

pra criar listtas bonitas usa a tag LI (listen item) abre com OL e entre OL usa o LI

  1. Acordar
  2. Abraçar a mãe
  3. Ficar compreguiça
  4. Viver o dia

LI no html 5 é opicional mas o OL ainda é obrigatório

        para listas em alfabeto use a tag A-a
        para listas em números use a tag OL 
        para listas em númeroos romanos use a tag I

listas não ordenadas

use a tag UL para criar listas sem ordem,mantendo a tag de abertura li

  • EXISTIR
  • ficar
  • nao sei
  • é isso aí guys
